About Todo App

A modern, responsive task management application built with the latest web technologies.

✨ Features

  • 📝 Create and manage tasks efficiently
  • ✅ Mark tasks as completed or pending
  • ✏️ Edit existing tasks inline
  • 🗑️ Delete unwanted tasks
  • 💾 Automatic data persistence with LocalStorage
  • 📱 Fully responsive design for all devices

🛠️ Technology Stack

Frontend
  • Angular 20
  • TypeScript
  • Bootstrap 5
  • SCSS
Architecture
  • Standalone Components
  • Reactive Forms
  • Angular Router
  • Service-based Architecture
